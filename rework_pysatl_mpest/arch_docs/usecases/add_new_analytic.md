# Добавление нового метода оценки параметров для конкретного распределения



Пусть реализован шаг оценки параметров, назовём его `SpecialStep` и реализована соответствующая общая стратегия оценки `special_step_estimate`. Тогда для конкретного распределения `ConcreteDistribution` необходимо реализовать конкретную стратегию `concrete_special_step_estimator`, реализующий логику оценки параметров. Необходимо учесть зафиксированные параметры и оценивать остальные параметры.
