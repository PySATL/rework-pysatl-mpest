# Добавление нового распределения



1. Создаем класс распределения, допустим `SpecialDistribution`, наследуясь от соответствующего типа распределения (например непрерывного). Необходимо:
   - Указать параметры через дескриптор, указав преобразования к внутреннему и внешнему представлению параметров (можно например через `lambda`), а так же указав ограничения параметра (например параметр `sigma` должен быть больше нуля)
   - Реализовать все методы интерфейса для ВНУТРЕННИХ параметров. Вся внутренная логика должна работать с внутренним представлением параметров.

2. Для данного распределения, реализуем (если это теоретически возможно) каждую конкретную стратегию оценки параметров. То есть, если представить таблицу Распределение/Шаг, в каждой ячейке которой лежит реализация конкретного алгоритма для конкретного распределения, заполняем строчку соответствующую `SpecialDistribution`.
